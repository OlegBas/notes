## Горячие клавиши (для VS Code и ОС)

1. F1 - run VS Code
2. F2 - Rename
3. ctrl+alt+1 - run Chrome
4. F4 - off computer
5. F5 - Reload Broser / Zen Mode
6. ctrl + / - Comment
7. F7 - toggle command line
8. F8 - toggle sidebar
9. ctrl+p search of files
10. ctrl+f - search of file
11. Show Todo Tree
12. Show Bookmarks
13. Ctrl+D - Copy Down
14. ctrl+h - замена
15. сtrl+g - переход к строке
