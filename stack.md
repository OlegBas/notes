## Горячие клавиши (для VS Code и ОС)

1. F1 - run VS Code
2. F2 - Rename
3. F3 - run Chrome
4. F4 - off computer
5. F5 - Reload Broser / Zen Mode
6. F6 - Comment 
7. F7 - toggle command line
8. F8 - toggle sidebar
9. F9 - copy (ctrl+c)
10. F10 - paste (ctrl+v)
10. F11 - search of files (ctrl+p)
11. F12 - search of file (ctrl+f)


12. Home - Show Todo Tree 
13. End - Show Bookmarks
14. Ctrl+D - Copy Down
